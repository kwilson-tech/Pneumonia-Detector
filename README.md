# Pneumonia-Detector

This is an image classifier for Chest X-Rays. Used the CNN model VGG16 to identify whether the given X-ray detects pneumonia.  

Libraries used:
* Tensorflow/Keras
* OS
* Numpy/Pandas/Matplotlib
* CV2
